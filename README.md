
# Web Development Course

A beginners course on web development technologies

**Start on March 24th in Berlin**

## Topics

1. HTML Basics
2. CSS Basics
3. include JavaScript sniplets
4. use the Bootstrap Framework
5. create a backend with Django
6. process forms and store data
