
# Building a Django Backend

**Django** is a professional web server written in the Python language.

In this course, we will follow the excellent [Django Girls Tutorial](https://tutorial.djangogirls.org/) that has been used by 26000+ participants and 1500 coaches.
In the tutorial, you will do the following:

- use a command line terminal
- start a Django server
- use Version Control
- deploy the server to a live page

For a more in-depth tutorial of Django, see the [Django Documentation](https://docs.djangoproject.com/en/6.0/).
