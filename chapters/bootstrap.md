
# Bootstrap

## What is Bootstrap?

Bootstrap is a **CSS Framework**. 
It is a set of prepared CSS classes that are easy to include in a web page to layout and format a page.
The purposes of using bootstrap are:

- avoid writing lots of CSS code from scratch
- get decent typography with little effort
- make the site more responsive on devices with different sizes

There are many other CSS frameworks like **Tailwind** or **Pico CSS*. Bootstrap is among the oldest and most popular ones.

## First steps with Bootstrap

1. go to [getbootstrap.com](httsp://getbootstrap.com)
2. go to the **Docs** section
3. from the example code in **getting started** you need to copy:

   - the viewport <meta> tag -> in <head>
   - the link to the CSS file -> in <head>
   - the link to the JavaScript file -> last thing in <body>

4. go to the documentation and pick an element
5. copy-paste it into your HTML

## Things to try

Build the following into your page using bootstrap classes:

- margins and padding
- a media item (image)
- a table
- a two-column layout
- buttons with links
- a caroussel (may require additional JavaScript)
