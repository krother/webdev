# JavaScript – Basics

## Goal

In this lesson, you will get a first taste of JavaScript and learn how to add interactive elements to your boat rental page by implementing an image carousel.

You will learn to:

- Understand what JavaScript is and why we use it
- Use the browser console for basic operations
- Work with variables to store data
- Connect a JavaScript file to your HTML
- Debug common JavaScript errors

## What is JavaScript?

**JavaScript (JS)** is a programming language that makes websites interactive and dynamic. While HTML provides structure and CSS provides styling, JavaScript adds behavior and interactivity.

JavaScript lets you:

- Respond to user actions (clicks, scrolling, typing)
- Change content on the page dynamically
- Create animations (note: CSS can do simple animations as well) and interactive elements
- Load and display data without refreshing the page

**Fun fact:** Despite the name, JavaScript has nothing to do with Java! It was named "JavaScript" as a marketing decision in 1995 to capitalize on Java's popularity at the time.

## Let's Explore: The Browser Console

Before we write JavaScript in a file, let's experiment in the browser console!

### How to open the console:

- **Chrome/Edge:** Right-click → Inspect → Console tab
- **Firefox:** Right-click → Inspect Element → Console tab
- **Keyboard shortcut:** F12 or Ctrl+Shift+J (Cmd+Option+J on Mac)

### Try these in the console:

**Simple math:**

```js
5 + 3;
10 * 2;
100 / 4;
```

### Storing values in variables:

```js
const myName = 'Your Name';
let boatCount = 5;

boatCount + 1;
```

### Printing variables

```js
console.log(myName);
console.log(boatCount);
console.log('I have ' + boatCount + ' boats');
```

## Exercise: Add an Image Carousel to Your Boat Rental Page

### Now let's add something interactive to your page - an automatic image carousel!

#### Step 1: Update your HTML

Add this carousel section to your HTML:

```html
<section class="carousel">
	<h1>Sakura Boat Rental</h1>
	<img
		id="carouselImage"
		src="sakura_boats.png"
		width="600px"
		alt="rowing boat on the cherry blossom lake"
	/>
	<!-- Dots to show which picture you're on -->
	<div class="dots" id="dotsContainer"></div>
</section>
```

### Step 2: Create you JavaScript file

Create a new file called carousel.js (or script.js) and copy the JavaScript code we debugged in class into this file.

### Step 3: Link your JavaScript to your HTML

Place your script tag in your head section, it should read sth like so

```js
<script src='script.js' defer></script>
```

### Step 4: Add your local images

```js
const images = [
	'boat1.jpg', // Replace with your actual image filenames
	'boat2.jpg',
	'boat3.jpg',
	'boat4.jpg',
];
```

Make sure:

- Your image files are in the same folder as your HTML file (or adjust the path accordingly)
- The filenames match exactly (including .jpg, .png, etc.)
- The spelling and capitalization are correct

### Challenge (Optional):
