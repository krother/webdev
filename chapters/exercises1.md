
# Recap Exercises

## Exercise 1: Single Page

Create a page on a topic of your choice that contains the following:

- a basic structure with ``<head>`` and ``<body>``
- multiple sections for content
- a header, footer or both
- headings
- some text

## Exercise 2: Sub-Pages

Create three subpages in addition to the main page.
Add the following:

- write a menu using a ``<nav>`` tag
- add links to all subpages and the main page inside the ``<nav>`` menu
- copy the menu to all four pages
- add an enumeration on one of the pages
- add a table on one of the pages
- add an image on one of the pages

## Exercise 3: CSS

Improve the look and feel of your page:

- create a CSS style sheet in a file ``style.css``
- link to the file in the head of your HTML pages
- add CSS classes to different sections of your page
- change the background color
- change the text color
- change the margins to have some spacing left and right

## Exercise 4: Flex

Use the flex layout to arrange elements in at least two places.
Here are two examples:

- create a menu column on the left and a content column on the right
- arrange the menu items horizontally

## Exercise 5: Caroussel

Add the caroussel as described in the JavaScript chapter.

## Reflect

Take notes:

- what worked well?

Show your result to the rest of the group.
